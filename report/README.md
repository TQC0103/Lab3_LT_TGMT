# Báo cáo Lab 03 - Thị giác máy tính

Template này kế thừa cấu trúc từ `Lab2-LT` và được tinh gọn để bắt đầu viết báo cáo mới nhanh hơn.

## Cấu trúc

```text
report/
|-- main.tex
|-- cover.tex
|-- preface.tex
|-- info.tex
|-- Glossary.tex
|-- references.bib
|-- assets/
|-- sections/
|-- figures/
|-- tables/
|-- appendix/
`-- build/
```

## Gợi ý sử dụng

1. Sửa các lệnh metadata ở đầu `main.tex`.
2. Viết nội dung tổng quan trong `preface.tex`.
3. Tổ chức nội dung chính bằng `sections/content_main.tex`.
4. Khi cần thuật ngữ viết tắt, bỏ comment dòng `\include{Glossary}` trong `main.tex`.
